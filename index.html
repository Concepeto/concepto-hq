Regla fundamental
Para cada pieza: generar una escena nueva en Higgsfield. Nunca pegar fotos encima de fondos. La escena generada ES el fondo.
1
Concepto + Frame Design
¿Qué quiero decir? ¿Qué principio psicológico aplico? ¿Qué layout uso (A–F)? ¿Cuál es la palabra NIVEL 1?
2
Escena Higgsfield
Modelo nano_banana_pro · MAX integrado en la escena, NO recortado · MAX corrido al lado donde NO va el texto · Foto de referencia: traje negro
3
Remotion — TipoReel
Escena como fondo full-bleed · Gradiente oscuro para legibilidad · Texto kinético palabra por palabra · Keyword highlight en #7c3aed · Hook en 1.5s
4
Virality Check + Publicar
Correr virality predictor de Higgsfield antes de publicar · Registrar métricas · Plataformas: Instagram feed, Reels, LinkedIn
Layouts a rotar
A · Visual grande, texto esquina
B · Visual pequeño, texto dominante
C · Equilibrio dinámico
D · Layout invertido
E · Editorial / magazine
F · Fondo en movimiento + texto
Nunca repetir el mismo layout dos veces seguidas. Si la anterior fue visual grande + fondo negro → la siguiente va texto dominante + fondo distinto.
